## PlayerAdmin

A PocketMine-MP Plugin | PlayerAdmin


Feature
------

* Warn system
* Instantly ban or ip-ban
* Can see player's address and DeviceOS, DeviceModel

|command|description|
|---|---|
|/playeradmin|PlayerAdmin's main command.
